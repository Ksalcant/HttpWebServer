You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html

    • Multiply Protocol

  • REQUEST:
  • /multiply?num1=3&num2=4 -- returns the value of the the multiplication
  • Path: multiply?
  • Body Parameter: num1(int) - required, you can type num1's value or num2(int)- optional
  • RESPONSE:
  • 200 OK
  • The Result is (result)
  • 500 Server error
  • Relax, those are values are too high for me still.
  • 400 Bad request!
  • 400: Wrong query, please try queries as follow: 'num1=3&num2=7'

    • Github Protocol

  • REQUEST:
  • /github?query=users/amehlhase316/repos -- returns the full_name, ids and ligin of the owner of each repo
  • Path: github?
  • Body Parameter: query(users/amehlhase316/repos) - required
  • RESPONSE:
  • 200 OK
  • json with names, ids and login info
  • 404 Bad request!
  • Your queries are misesplled, please fix them and try again

    • Weather Protocol

  • REQUEST:
  • /weather?city1=paterson&city2=Kansas -- returns the weather of one city or two cities at the same time.
  • Path: weather?
  • Body Parameter: city1(String) - required, you can type any city as the value for city1 or city2(String) - optional
  • RESPONSE:
  • 200 OK
  • The current weather in paterson is: Clear
  • The current weather in Kansas is: Clear
  • 400 Bad request
  • An error occurred while processing the request for weather.
  • You need to add 'city1=nameOfCity1&city2=nameOfCity2'

    • Palindrome Protocol

  • REQUEST:
  • /palindrome?str1=ABA&str2=Abichuelas - returns a text saying if any of the string are palindrome. You can check one string or two.
  • Path: palindrome?
  • Body Parameter: str1(String) - required, you can type any string as the value for str1 or str2(String) - optional
  • RESPONSE:
  • 200 OK
  • (str) is palindrome
  • (str) is not palindrome
  • 400 Bad request
  • Error1: An error occurred while processing the request for palindrome.
  • 404 Bad request
  • Error2: Missing 'str1' and 'str2' parameters in the request for palindrome.'
